# Client social network

## ⚙️ Technologies

- react.js
- redux.js
- redux-form
- axios

## 🚀 Installation

```shell
cd client
npm install
npm run start
```

### 🎉 The Client started on [http://localhost:3000](http://localhost:3000)

The page will reload if you make edits.\
You will also see any lint errors in the console.
